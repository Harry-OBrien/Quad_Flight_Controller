Detailed Description

A structure to describe the physical state of the pod, whether that is a target state or a current state

Abbreviations: s: dispacement (m) v: velocity (m/s) a: acceleration (m/s^2) t: theta (attitude/angle) (radians) g: delta theta (rotational velocity) (radians/s)
